Open Source

Frameworks & Libraries

Every JavaScript library and CSS framework used in this project. All are open-source and free to use.

  • Tailwind CSS (CDN) — MIT License  ·  cdn.tailwindcss.com  ·  Utility-first CSS framework used for responsive layout, spacing, and styling throughout the site.
  • Leaflet 1.9.4 — BSD-2-Clause License  ·  leafletjs.com  ·  Interactive map rendering for the resource directory map view and resource detail mini-maps.
  • Leaflet MarkerCluster 1.5.3 — MIT License  ·  github.com/Leaflet/Leaflet.markercluster  ·  Smart clustering of map markers when zoomed out so overlapping pins group together cleanly.
  • Leaflet Heat 0.2.0 — BSD-2-Clause License  ·  github.com/Leaflet/Leaflet.heat  ·  Density heatmap overlay on the resource map showing where resources are concentrated.
  • Apache ECharts 5 — Apache-2.0 License  ·  echarts.apache.org  ·  2D data visualizations including radar chart, sankey flow diagram, and live community metrics dashboard.
  • ECharts-GL 2 — BSD-3-Clause License  ·  github.com/ecomfe/echarts-gl  ·  WebGL extension for ECharts enabling 3D bar chart rendering in the analytics section.
  • Three.js r128 — MIT License  ·  threejs.org  ·  3D rendering engine used for the community orbit visualization and first-person virtual city world explorer.
  • AOS 2.3.1 — MIT License  ·  michalsnik.github.io/aos  ·  Animate On Scroll library that triggers smooth entrance animations as users scroll down the page.
  • Lucide Icons — ISC License  ·  lucide.dev  ·  Consistent SVG icon library used for all interface icons across the site — navigation, cards, buttons, and more.

Audio & Sound Effects

All sound effects and audio clips used throughout ConnectHBG were created entirely by the project team members (KA & YV). No third-party audio libraries, stock sound sites, or copyrighted recordings were used. All audio is original work and is not subject to any external license.

  • UI Sound Effects — Original — created by KA & YV  ·  All click, notification, confirmation, and interaction sounds used in the site interface were produced by the project team. No external audio sources were used.
  • Text-to-Speech Audio — Device built-in feature  ·  The text-to-speech voice heard when using the accessibility toolbar is generated entirely by the operating system's built-in speech engine (present on all Windows, macOS, iOS, and Android devices). No audio files were recorded or imported for this feature.

Web Standards

APIs & Browser Features

  • Web Speech API — W3C Specification  ·  Built into modern browsers  ·  Used to power the voice search feature in the navigation bar. Converts spoken words into text search queries without any third-party service.
  • SpeechSynthesis API — W3C Specification  ·  Built into all modern devices and operating systems (Windows, macOS, iOS, Android)  ·  Powers the text-to-speech accessibility feature in the toolbar. Uses the device's own built-in speech engine — no external audio files or third-party services involved.
  • localStorage / sessionStorage — W3C Web Storage Specification  ·  Built into all modern browsers  ·  Used to persist user preferences (dark mode, font size, favorites, login session) across page loads and sessions without a backend server.
  • Pointer Lock API — W3C Specification  ·  Built into modern browsers  ·  Used by the Three.js PointerLockControls to capture mouse input for first-person camera movement in the 3D Community City explorer.

Project Stack

  • HTML5 / CSS3 — Semantic page structure, accessibility roles (ARIA), responsive layout, custom properties (CSS variables), and animation.
  • JavaScript (ES6+) — All site logic — filtering, searching, rendering, dark mode, modal management, user accounts, and more. No build step or bundler required.
  • Tailwind CSS — Utility-first CSS framework loaded via CDN for rapid responsive styling without writing custom CSS for every component.
  • Leaflet.js — Interactive map layer with MarkerCluster for grouped pins and Leaflet Heat for density heatmap visualization of resource locations.
  • Apache ECharts — 2D data visualization library powering the community analytics section — radar chart, sankey diagram, and live metric charts.
  • ECharts-GL — WebGL extension for ECharts enabling GPU-accelerated 3D bar charts in the community analytics dashboard.
  • Three.js — 3D graphics library rendering the community orbit globe visualization and the first-person 3D Community City explorer with WASD movement and pointer lock.
  • AOS (Animate On Scroll) — Scroll-triggered animation library that fades and slides content into view as users scroll, providing a dynamic and polished feel.
  • Lucide Icons — Consistent, accessible SVG icon set providing over 1,000 pixel-perfect icons used throughout navigation, cards, buttons, and all UI elements.
  • Inter Typeface — Google Fonts-hosted variable font designed for screen legibility. Used at weights 300–900 across all text on the site.
  • LocalStorage / SessionStorage — Browser-native key-value storage used to persist dark mode, font size, favorites, and user account session without any backend or server.
  • Web Speech & SpeechSynthesis APIs — Native browser APIs enabling voice search input and text-to-speech accessibility output, making the site more inclusive for all users.
